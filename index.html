Section 01

Strategic Vision

The ICA website today serves mostly as a static information page. This proposal outlines how to grow it into a working platform — one that supports program registration, family communication, and daily school operations in one place.

"The school website will serve as ICA's central hub for communications and programs — making it easier for parents to get information, improving event and program participation, and giving administration clear data on what's working."

Three core shifts define this transformation:

Static → Operational

From brochure to platform

Manage program registrations, communications, and student and parent resources all in one place.

Manual → Automated

Less repetitive work

Social posts, event reminders, and email updates run on schedules — reducing workload for staff.

Invisible → Measurable

Clear performance data

Every major component tracks real KPIs — page visits, event sign-ups, email opens — so decisions are based on data, not guesswork.

Section 03

Core Program Microsites

Each major school program gets its own page — a branded landing area under the main ICA domain. The Summer Game Dev Camp is the first one, and it becomes the template for everything that follows: camps, athletics, clubs, and student programs.

Summer Game Dev Camp Microsite

The existing Summer Game Dev Camp page moves into the ICA website as the first fully tracked program microsite. Everything built here — the design, the registration flow, the analytics — gets reused for every future program page.

  • Program overview, schedule, and instructor profiles
  • Online registration with automatic confirmation emails
  • Student project gallery and highlights from past cohorts
  • FAQ section and parent information package
  • Social sharing panel (LINE, Facebook, Instagram)
  • KPI tracking: page visits → registrations → completions

Future Program Microsites

AI & Machine Learning Camp

A 1–2 week program covering Python, data basics, and hands-on ML projects using Jupyter and Colab — built on the AI tools already in use.

Web Development Workshop

HTML, CSS, JavaScript, and GitHub Pages. Students leave with a live portfolio site — a natural extension of the CS curriculum.

AP Cybersecurity Prep Camp

Grounded in Taiwan critical infrastructure awareness. Covers ethical hacking, network defense, and digital safety using the existing AP Cyber curriculum.

Senior Thesis Defense Program

A portal for students, parents, and evaluators — with submission deadlines, defense schedules, evaluator assignments, and an archive of past projects.

Section 04

CS Program Microsite

The Computer Science and Computational Mathematics program at ICA runs on tools no other school in Taiwan uses — GitHub Codespaces, VS Code, Docker, Jupyter, Python, and a custom grading pipeline handling 500+ student repositories. That's a real differentiator, and it deserves its own page.

GitHub / VS Code Curriculum Hub

A public-facing hub that shows what the program actually looks like — its philosophy, student outcomes, and the tools students use from day one. This turns an invisible internal system into something families and prospective students can see and appreciate.

  • Program philosophy: project-based learning with industry-standard tools
  • Live student repo statistics (524+ active repositories)
  • Tools showcase: GitHub Codespaces, VS Code, Docker, Jupyter, Python
  • Student work gallery — published projects via GitHub Pages
  • Curriculum path: Algebra 2 → Computational Math → Coding → AP CS
  • Faculty spotlight and teaching philosophy
  • Industry alignment section to build parent confidence
Section 05

LMS & Course Hub

Each course gets its own home on the website — a place where students, parents, and co-teachers can find assignments, syllabi, and resources without hunting through email or messaging apps. These pages connect to tools already in use like Google Workspace, GitHub Codespaces, and Jupyter.

Course Portals

Algebra 2

Weekly lesson summaries, downloadable worksheets, unit calendars, and links to Khan Academy. Parents can see upcoming tests and recent scores without a trip to the office.

Computational Math Hub

Jupyter notebook assignments, Python lab documentation, module rubrics, and a student project archive — with direct links to GitHub Codespaces for live environment access.

Coding Class Portal

GitHub Classroom assignment links, VS Code setup guides, Docker documentation, curriculum roadmap, and a student portfolio showcase pulled from GitHub Pages.

Section 06

Reusable Program Framework

Beyond academics and camps, every school program — athletics, clubs, student government, international exchange — gets its own section of the ICA website, all built from the same modular template.

Athletics Hub

Basketball team roster, game schedule, score recaps, coaching profiles, and a photo/video gallery. Game results sync automatically. Highlights team discipline and student leadership.

Clubs Directory

A searchable list of all student clubs with meeting times, faculty sponsors, member counts, and a simple sign-up form — replacing word-of-mouth with something parents and students can actually find.

Student Government Portal

Meeting minutes, school initiatives, student polls, and a submission channel for student proposals — building a visible culture of student participation.

International Exchange Hub

Partner school directory, MOU listings, delegation archives, and student exchange stories — linked to Taiwan's Ministry of Education International Exchange Window.

Senior Thesis Archive

A public archive of completed thesis projects. Students submit work, track defense schedules, and browse past papers. Evaluator assignments are handled through the staff backend.

Events & Calendar Center

One unified school calendar with online registration and payment for field trips, after-school programs, performances, and summer camps — no more paper forms.

Section 07

Social Media & Automation

The website becomes the source of record for all school content. Social channels — Facebook, Instagram, YouTube, LINE — act as the distribution layer, fed by automated workflows that bring manual posting time close to zero.

Publish to the website first → auto-generate social drafts → schedule posts → route comments and messages to a shared inbox → track referral traffic back to the site.

Integration Tiers

Static Integration

Social follow buttons, Instagram feed on the homepage, YouTube playlist embeds on program pages, and share buttons on news and event posts. Manual posting continues, but everything links back to the website.

Semi-Automated

Social post drafts are created automatically when a news or event page is published. A social scheduler queues them for review before they go live. Event reminder posts are generated 7 days and 1 day before each event.

Full Automation

A workflow automation tool connects the CMS to social posts, sends LINE messages to parents, generates video captions, and routes all comments and DMs to a shared inbox. A monthly analytics report is generated automatically.

Section 08

Administrative Efficiency

Every page that answers a question a parent or student would otherwise come to the office for is a real reduction in workload. These self-service modules deliver the highest return on effort.

Parent Self-Service Portal

Attendance records, grade summaries, upcoming test dates, fee schedules, school policies, and teacher contact forms — all in one place, no office visit needed. Studies show self-service portals cut repeat administrative inquiries by 50–60%.

Emergency Notification Center

A dedicated section for school closure notices, weather alerts, and urgent updates. Publishing once automatically updates the website, sends a newsletter, posts to LINE, and publishes on Facebook — one action, four channels.

Forms & Documents Center

Permission slips, medical forms, club sign-up sheets, fee records, and official school documents — all downloadable and submittable online. No more paper circulation or lost-form follow-ups.

Monthly Communications Dashboard

An internal analytics page showing website traffic, email performance, social reach, and video views together. Auto-generated monthly via Google Analytics and Looker Studio — no manual reporting needed.

Section 09

Family Engagement

A school website that families come back to — not just bookmark and forget — needs content that's fresh, personally relevant, and available in the right language. Here's what keeps parents connected between parent-teacher conferences.

Student Life Gallery

Weekly photos and video updates from classrooms, events, and programs. Auto-feeds to Instagram and Facebook.

Bilingual Content (EN/ZH)

Key pages — calendar, policies, program info — available in both English and Traditional Chinese. Essential for accessibility in a bilingual school community like ICA.

Teacher Spotlight Blog

Short monthly posts from teachers about their classes, projects, and what they've been working on. Builds trust with families while generating fresh content for search and social sharing.

Community Events Board

A calendar that goes beyond school events — including local family-relevant dates like college application deadlines, SAT dates, scholarship announcements, and community gatherings.

Newsletter Sign-up Hub

One-click newsletter sign-up on every major page. Parents choose their preferences: weekly summary, program updates, emergency alerts only, or everything.

Student Achievement Showcase

A public page celebrating student wins — competition results, project completions, and GitHub portfolio milestones. Visible, shareable recognition instead of private ceremonies.

Next Steps

Three concrete steps to move from proposal to launch:

  1. Administration Approval

    Present this proposal to school leadership and the IT committee. Identify a faculty web coordinator and confirm the Phase 1 approach — most of which relies on tools ICA already has access to.

  2. Summer Camp Pilot

    Bring the existing Game Dev Camp GitHub Pages site into the ICA domain as the first branded program microsite. Set up analytics before Phase 2 starts in August so we have a clear baseline.

  3. CS Microsite Build

    Build the CS Program Hub as the internal flagship — showing student repos, tools, and the curriculum to families. This is ICA's strongest differentiator and costs nothing beyond staff time to build.
